# 🎬 Movie Explorer

A modern, responsive movie discovery web application built with **React** and **Vite**. Browse popular movies, search your favorites, and enjoy a clean, intuitive user interface designed for movie enthusiasts.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/movie-explorer.git
```

Go to the project folder

```bash
cd movie-explorer
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
