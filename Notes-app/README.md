# 📝 React Notes App

A modern and responsive Notes Application built with **React.js**, **Vite**, and **Tailwind CSS**. The app allows users to create, pin, unpin, and delete notes with a clean, intuitive interface for organizing important information.

![React](https://img.shields.io/badge/React-19-61DAFB?logo=react)
![Vite](https://img.shields.io/badge/Vite-7-646CFF?logo=vite)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-4-38BDF8?logo=tailwindcss)
![License](https://img.shields.io/badge/License-MIT-green)

---

## ✨ Features

- ➕ Create notes with title and description
- 📌 Pin & unpin important notes
- 🗑️ Delete notes instantly
- 📂 Automatic separation of pinned and regular notes
- 📱 Fully responsive design
- ⚡ Fast UI powered by React Hooks
- 🎨 Modern dark theme using Tailwind CSS
- ✨ Empty state when no notes are available

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Dhawal-Bhawsar/react-notes-app.git
```

Move into the project

```bash
cd react-notes-app
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:5173
```

## 💡 How It Works

1. Enter a note title.
2. Add the note description.
3. Click **Add Note**.
4. Pin important notes to keep them at the top.
5. Delete notes whenever they are no longer needed.
